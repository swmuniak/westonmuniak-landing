This site is configured for easy deployment to GitHub Pages:

1. Push your changes to the `main` branch
2. Run `npm run deploy` to build and deploy
3. Your site will be live at `https://yourusername.github.io/weston-muniak-landing/`

### Custom Domain

To use a custom domain:

1. Add a `CNAME` file to the `public` folder with your domain
2. Configure your domain's DNS to point to `yourusername.github.io`
3. Enable custom domain in your GitHub repo settings
